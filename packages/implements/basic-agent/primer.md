# Basic Agent Primer

> **Для ИИ-агента**: Default implementation of AGENT concept

---

## Что это?

**BASIC_AGENT** — дефолтная реализация концепта AGENT.

Работает везде, используя доступные инструменты или приближённую оценку.

---

## Методы

### get_info() → AgentInfo

Верни информацию о себе:
- Если знаешь свою модель — используй это
- Иначе — проверь `.cm/project.yml`
- Иначе — используй дефолты

```yaml
{ model: "claude-sonnet-4", provider: "anthropic", context_window: 200000 }
```

### get_tokens_used() → number

Оцени использованные токены:
- Используй инструменты подсчёта если доступны
- Иначе — подсчитай символы и раздели на 4
- Добавь ~1000 токенов на системные промпты

### get_tokens_available() → number

```
context_window - get_tokens_used()
```

### estimate_tokens(text) → number

Оцени токены в тексте:
- Английский: ~4 символа = 1 токен
- Код: ~3 символа = 1 токен
- Кириллица/CJK: ~2 символа = 1 токен

### can_fit(tokens) → boolean

```
get_tokens_available() >= tokens
```

---

## Важно

1. **Всё приблизительно** — это нормально
2. **Лучше переоценить** — безопаснее думать, что токенов меньше
3. **Не блокируй** — всегда давай ответ, даже если неточный
